# Skill Galaxy Hackathon

Vite + React scaffold for a talent profile and opportunity platform.

## Project structure

```text
src/
	components/
		Auth/
		Opportunities/
		Profile/
		Verification/
	firebase/
	hooks/
```

## Included placeholders

- Login and registration forms
- Profile creation, skills, and evidence upload sections
- QR verification and profile sharing blocks
- Opportunity feed and reusable card component
- Firebase config module and auth hook

## Environment variables

Set Firebase values in `.env`:

```env
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
```

## Commands

```bash
npm install
npm run dev
npm run build
```
